## Deploy

```sh
git clone https://github.com/FayasNoushad/Short-Link-API/tree/main
cd Short-Link-API
pip3 install -r requirements.txt
python3 app.py
```

---

## Usage

```
https://shortlink.api.fayas.me/?query=<your-link>
```
